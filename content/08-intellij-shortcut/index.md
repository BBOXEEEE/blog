## ✨ IntelliJ 단축키를 정리해보자!
사실.. 나는 단축키를 잘 사용하지 못한다🤣🤣🤣🤣 <br>
얼마전 스프링 공부를 위해 IntelliJ IDE를 사용하면서 단축키가 진짜 편리하다는 것을 느꼈다. 그동안은 이클립스를 주로 사용했는데 이클립스에서도 단축키를 잘 사용한 것 같지 않은데 인텔리제이를 쓰면서 단축키를 조금씩 조금씩 사용하면서 와.. 진짜 편하다는 것을 느꼈다.<br>
그래서! 자주 사용하지 않으면 까먹으니까, 까먹을 때마다 검색하기 귀찮으니까, 단축키를 정리해보도록 하겠다!

## 👋 시작하기 전에
나는 macOS를 사용한다. 그러므로, macOS 기준으로 단축키를 정리할 것이다. 매번 기호를 적기 귀찮으니까 아래와 같이 명시해두겠다.
> ⌘ : Command <br>
> ⌃ : Control <br>
> ⌥ : Option <br>
> ⇧ : Shift

## 🗳️ IntelliJ 단축키 모음
### ⚙️ General
- Command + 1~9 : 각 도구창에 대한 토글 <br>
|단축키|기능|
|:-----:|:-----:|
|Command + 1|Project|
|Command + 2|Favorite|
|Command + 3|Search|
|Command + 4|Run|
|Command + 5|Debug|
|Command + 6|Problems|
|Command + 7|Structure|
|Command + 8|Services|
|Command + 9|VCS|
|Command + ,|Settings, 설정|
|Command + ;|Project Structure, 프로젝트 설정|
|Command + Shift + F12|편집기 영역 최대화|
|Option + F12|Terminal|

### 📌 자주 쓰는, 꼭 기억해야하는 단축키
|단축키|기능|
|:-----:|:-----:|
|Control + Shift + Space|스마트 코드 완성|
|Option + Enter|Quick Fix|
|Command + N|Generate Code (생성자, getter & setter, override 등)|
|Option + ↑|확장 영역이 점점 확장|
|Option + ↓|확장 영역이 점점 축소|
|Command + E|최근 작업 파일 목록|
|Shift + F6|Rename, 참조되는 파일 이름 전부 변경|
|Shift + Shift|모두 검색|

### 💨 Navigation
|단축키|기능|
|:-----:|:-----:|
|Command + B|해당 코드 정의 부분으로 이동|
|Command + Shift + B|해동 코드의 구현부로 이동|
|Command + O|클래스로 이동|
|Command + Option + O|메소드로 이동|
|Command + U|슈퍼 메소드/ 슈퍼 클래스로 이동|
|F2 or Shift + F2|오류 또는 경고 위치로 이동|
|Command + L|지정한 라인으로 이동|

### ✒️ Editor
|단축키|기능|
|:-----:|:-----:|
|Command + D|라인 복사|
|Command + Delete|라인 전체 삭제|
|Control + Space|기본 코드 완성|
|Command + Shift + Enter|세미콜론으로 끝나면 문장의 끝으로 이동, 아니면 한 줄 엔터|
|Command + /|한 줄 주석 코드|
|Command + Option + /|블록 단위 주석 토글|
|Command + Option + L|Reformat Code (적용 영역 선택)|
|Control + Option + O|Import 최적화|
|Control + W|현재 에디터 탭 닫기|

### 🔮 Refactoring
|단축키|기능|
|:-----:|:-----:|
|F5|파일 복사|
|F6|파일 이동|
|Control + T|리팩토링 관련 전체 항목 조회|
|Command + F6|파라미터, 접근제어자, 이름, 리턴타입 등을 편집|
|Command + Delete|Safe Delete|
|Command + Option + N|메소드 인라인 처리|

### ↪️ Compile & Run
|단축키|기능|
|:-----:|:-----:|
|Command + F9|프로젝트 빌드|
|Control + R or D|Run or Debug|
|Option + Shift + R|실행|
|Command + F2|실행 종료|
|Command + F8|브레이크 포인트 설정|

### 🌈 Live Template
|단축키|기능|
|:-----:|:-----:|
|Command + J|Live Template 목록 조회|
|sout|System.out.println()|
|fori|for 문|
|iter|for-each 문|
|psf|public static final|
|prsf|private static final|
|psvm|public static void main(String[] args)|
|thr|throw new|
